How "Login with Site Phantom" Works

  1. Register your app below to get a Client ID and Client Secret.
  2. Redirect users to: 
    {{ base_url }}/oauth/authorize?client_id=YOUR_CLIENT_ID&redirect_uri=YOUR_REDIRECT_URI&response_type=code&scope=identify
  3. After the user approves, they'll be redirected to your redirect_uri with a ?code=... parameter.
  4. Exchange the code for an access token by POSTing to: 
    POST {{ base_url }}/oauth/token
Content-Type: application/x-www-form-urlencoded

grant_type=authorization_code&code=THE_CODE&redirect_uri=YOUR_REDIRECT_URI&client_id=YOUR_CLIENT_ID&client_secret=YOUR_CLIENT_SECRET
  5. Use the token to fetch user info: 
    GET {{ base_url }}/oauth/userinfo
Authorization: Bearer ACCESS_TOKEN
    Returns: { "id", "username", "discord_id", "roblox_id" }
